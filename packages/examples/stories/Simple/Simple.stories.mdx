# Simple Component Example

## MDX

Importing `A11yDocsBlock` and configuring directly in MDX.

```tsx
import { A11yDocsBlock } from "@abcaustralia/storybook-addon-a11ydocs/storybook-addon-a11ydocs";
```

<A11yDocsBlock
  customProperties={{
    ...a11ydocs.primary,
    ...a11ydocs.secondary,
  }}
/>
